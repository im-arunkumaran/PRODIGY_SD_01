This is a simple Temperature Converter application built using Python and Tkinter.
It allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin using a clean black and white graphical interface.

The goal of this project was to practice:

1.Basic Python functions 
2.GUI development using Tkinter 
3.Handling user input and exceptions
4.Improving UI appearance and resolution

How to Run:

1.Make sure Python 3 is installed.
2.Download or clone this repository. 
3.Open the project folder.
4.Run the file
